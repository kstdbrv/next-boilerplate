# Next.js with TypeScript, ESLint, Prettier and Commitlint Jira

Starter code for a clean Next.js + TypeScript + ESLint + Prettier and Commitlint Jira project.

## Get started

```sh
# Install dependencies
yarn install

# Enable husky
yarn husky install

# Start dev server
yarn dev
```
